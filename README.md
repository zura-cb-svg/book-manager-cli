# 📚 Book Manager CLI

A simple command-line application to manage books using Python.

---

## 🚀 Features

* Add book
* Show books
* Search books
* Delete book
* Update book
* Count books by author
* Sort books by year

---

## 🧱 Project Structure

```
book-manager-cli/
│
├── main.py
├── models/
│   └── book.py
├── manager/
│   └── book_manager.py
├── utils/
│   └── input_utils.py
```

---

## 💾 Data Storage

Books are saved in a JSON file (`books.json`).

---

## ▶️ How to Run

```
python main.py
```
